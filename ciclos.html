Ciclos

Los ciclos nos sirven para ejecutar un bloque de código varias veces. De esta manera no tenemos que escribir las mismas líneas de código n veces.

Hay distintos tipos de ciclos y según la situación nos convendrá usar uno u otro. Existen el ciclo for y el ciclo while.

El ciclo while se ejecutará siempre que la condición que indiquemos sea verdadera. Por otro lado, el ciclo for iterará un proceso el número de veces que nosotros queramos.

Vemos cómo funcionan estos dos ciclos.

Ciclo for

Empecemos con el ciclo for. Imaginemos que queremos sumar los primeros 50 números naturales (sin el cero). Para esto usaremos la función range(b, c). Esta función es incluyente en el extremo izquierdo y excluyente en el extremo derecho. Es decir, range(b, c) recorrerá todos los números mayores o iguales a b y menores estrictos a c. 
suma = 0
for i in range(1, 51): # números del 1 al 50, el 51 no se incluye
    suma = suma + i # esta línea de código se ejecuta 50 veces
print(suma) # 1275

Ciclo while

Analicemos ahora el ciclo while. Para ejecutar un ciclo while debemos establecer una condición que le indique al código cuándo entrar al ciclo y cuándo salirse. Mientras la condición sea verdadera, el bloque de código dentro del ciclo se va a ejecutar.

Veamos un ejemplo en el que le preguntamos al usuario si quiere salir del programa o no. Mientras el usuario nos diga que no, le seguiremos preguntando si desea salir. En este caso no sabemos cuántas veces el usuario nos va a decir que no, por lo tanto, un ciclo for no nos serviría.

salir = 'no'
while salir == 'no': # revisamos si la condición es verdadera
    salir = input('¿Deseas salir del programa? ')
print('Hemos salido del ciclo.')
