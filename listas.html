Listas

Las listas en Python son muy útiles para guardar información preservando su orden. Se guardan entre corchetes [ ] y sus elementos pueden ser cualquier tipo de variable. Los elementos de una lista están indexados empezando en 0.

Para acceder a un valor específico de una lista, indicamos el índice que queremos entre corchetes. Por ejemplo, si buscamos el tercer valor de la lista llamada lista lo indicamos así: lista[2]. Recuerda que los índices empiezan en 0. Por eso el índice del tercer elemento es 2 y no 3.

También podemos ir de atrás hacia adelante con números negativos. Por ejemplo, para acceder al último elemento de una lista podemos usar el índice -1 de esta manera: lista[-1].

Veamos algunas funciones comunes de las listas. 
lista = ['hola', 71, 6.9]
print(lista[0])# 'hola'
print(lista[-1])# 6.9
lista.append('bye')
print(lista)# ['hola', 71, 6.9, 'bye']
print(lista[-1])# 'bye'
lista.pop(1)# quita el elemento con índice 1 de la lista, en este caso el 71
print(lista)# ['hola', 6.9, 'bye']
Podemos iterar sobre los elementos de una lista. 
lista = [5, 10, 15, 20]
cuadrados = []
for i in lista:	
    cuadrados.append(i**2)
print(cuadrados)# [25, 100, 225, 400]
Para insertar un elemento a una lista en una posición específica podemos usar el método insert. 
lista = [1, 2, 4, 5]
lista.insert(2, 3)# insertamos el número 3 en la posición 2
print(lista)# [1, 2, 3, 4, 5]
lista.append(6)# agrega el número 6 hasta el final de la lista
print(lista)# [1, 2, 3, 4, 5, 6]
