Condicionales

Los condicionales nos sirven para ejecutar un bloque de código en caso de que se cumpla cierta condición. En caso contrario podemos ejecutar un código distinto o no ejecutar nada.

Para esto podemos usar un if else. Sin embargo, cuando queremos evaluar varias condiciones distintas podemos usar un elif en vez de un else.

Veamos algunos ejemplos.

Un clásico if else para ver si un número es par o impar. En este ejemplo, se va a evaluar la condición n%2 == 0. En caso de que la condición sea verdadera, el número es par. En otro caso, el número es impar. 
n = 27
if n%2 == 0:
    print(f'{n} es par.')
else:
    print(f'{n} es impar.')
Veamos un ejemplo de un if elif else. Queremos ver si un número es estrictamente positivo, negativo o cero. 
n = -13
if n > 0:
    print(f'{n} es estrictamente positivo.')
elif n < 0:
    print(f'{n} es negativo.')
else:
    print(f'{n} es cero.')

Es importante mencionar que cuando usamos un if elif else, la condición que se cumpla primero es la que se va a ejecutar y las demás condiciones ya ni serán revisadas.

Veamos el siguiente ejemplo en el que evaluamos si un número es par o múltiplo de 3. Como la primera condición que se revisa es si el número es par, los números que son pares y múltiplos de 3 sólo entrarán a la primera condición. 

n = 18
if n%2 == 0: # verdadero
    print(f'{n} es múltiplo de 2.') # esto se va a imprimir
elif n%3 == 0: # esta condición es verdadera, pero ya no se revisa porque la condición anterior se cumplió
    print(f'{n} es múltiplo de 3.') # esto no se va a ejecutar
else:
    print(f'{n} no es ni múltiplo de 2 ni de 3.')
Un if no necesariamente tiene que tener un elif o un else. Si no se cumple la condición que queremos, podemos simplemente no hacer nada. Veamos un ejemplo en el que nos interesa sacar el valor absoluto de un número. 
n = 93
if n < 0:
    n = -n # esta línea de código no se va a ejecutar.
print(n) # 93
