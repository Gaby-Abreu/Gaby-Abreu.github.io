Try Except

El try except nos permite cachar posibles errores en el código para evitar que nuestro programa explote. Nos da la oportunidad de intentar correr un bloque de código y, en caso de que no funcione, intentar algo distinto.

Veamos algunos ejemplos.

Si tratáramos de convertir el string '7.9' a un número entero, el código nos marcaría un error, pues 7.9 no es un número entero. Sin embargo, si usamos un try except podemos primero intentar convertirlo a entero y, en caso de no ser posible, convertirlo a un float. 
b = '7.9'
try:
    c = int(b)
except:
    c = float(b)
print(c)#7.9

El try except nos permite adelantarnos a posibles errores que pueda cometer el usuario. Veamos el siguiente ejemplo en el que le pedimos un número entero al usuario. Si el usuario nos da otro tipo de variable, le volvemos a pedir un número.

En este caso no depende de nosotros que el usuario ingrese lo que le pedimos de forma correcta. Por eso es bueno tener una manera de evitar posibles errores que puedan hacer que nuestro código ser rompa.

while True:
    n = input('Dame un número entero: ')
    try:
        n = int(n)# tratamos de convertir n a un número entero
        break# en caso de lograrlo nos salimos del ciclo
    except:
        print('Eso no es un número entero.')# caso en el que n no es un número entero
